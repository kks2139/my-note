# my-note - Making my own notes
자유롭게 메모하고 저장할 수 있는 SPA 구현

***

### 화면 개요
1. 로그인 / 회원가입
2. 메인 
    - 세션 정보
    - 필기 섹션 리스트
    - 필기 영역
    - 입력 옵션 (폰트크기, 스타일, 그리기 도구 등)
3. 잘못된 URL
    - 404 Not Found


### 기술 스택
1. Server : ExpressJS / AWS RDS / MySQL
2. Client : React Hooks / Typescript / Javascript / HTML / CSS
